shift_holder_app.py - macOS 메뉴바 Shift 홀드 매크로
=====================================================

개요
----
macOS 메뉴바에서 실행되는 자동 키 입력 매크로입니다.
Shift 키를 일정 시간 홀드한 뒤, 방향키 입력을 패턴에 따라 반복합니다.

의존성
------
  pip install rumps pyautogui pynput

실행
----
  python shift_holder_app.py

  macOS 접근성 권한(손쉬운 사용)이 필요합니다.
  시스템 환경설정 > 개인 정보 보호 및 보안 > 손쉬운 사용에서 터미널/앱 허용


패턴 목록
---------
  패턴 A [1]  좌 10s / 우 3s   (홀수 회차: 좌, 짝수 회차: 우)
  패턴 B [2]  우 10s / 좌 3s   (홀수 회차: 우, 짝수 회차: 좌)
  패턴 C [3]  좌 10s / 우 10s  (홀수 회차: 좌, 짝수 회차: 우, 동일 홀드)
  패턴 D [4]  Shift 30~40s → 좌+우  (1/3회차: 좌→우, 2·3회차: 우)
  패턴 E [5]  Shift 35s → 좌  (매 회차 좌 방향 1회)

단축키
------
  1  패턴 A로 전환 (실행 중이면 재시작)
  2  패턴 B로 전환
  3  패턴 C로 전환
  4  패턴 D로 전환
  5  패턴 E로 전환
  6  정지

  메뉴바 아이콘 클릭 후 [▶ 시작 / ⏹ 정지] 으로도 제어 가능합니다.

동작 흐름 (매 회차)
-------------------
  1. Shift 키 홀드 (패턴에 따라 고정 또는 랜덤 시간)
  2. Shift 해제 후 z 키 입력
  3. 패턴에 따라 방향키 입력
  4. 반복

상수 (shift_holder_app.py 상단에서 조정)
-----------------------------------------
  HOLD_MIN_ODD / HOLD_MAX_ODD  패턴 A·B 홀수 회차 홀드 범위 (초)
  HOLD_MIN_EVEN / HOLD_MAX_EVEN 패턴 A·B 짝수 회차 홀드 범위 (초)
  HOLD_C                        패턴 C 홀드 시간 (초)
  HOLD_D_MIN / HOLD_D_MAX       패턴 D 홀드 범위 (초)
  HOLD_E                        패턴 E 홀드 시간 (초)
